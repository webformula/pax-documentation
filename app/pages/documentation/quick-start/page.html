Installation

You can install pax-core through NPM

// The only required package npm i @webformula/pax-core --save // install these to get a full development environment started // you will need these for this quickstart npm i snowpack nodemon --save-dev

PAX build

This is a optional build step that will auto generate you entry file and compile your html into a single template cache

Create pax build config buildEntry.mjs

import buildEntry from '@webformula/pax-core/src/buildEntry.js'; buildEntry({ rootFolder: 'app', pagesFolder: 'pages', templateFile: true, routerConfig: { root: 'Home' } });

Nodemon

Auto rebuild all the things whenever any file changes

Create nodemon config Nodemon.json

{ "watch": [ "app/" ], "ext": "js, css, html", "ignore": [ "app/pax-entry.js", "app/pax-templates.js" ] }

Snowpack

Snowpack is a great build tool that can make development and building much faster and simpler

Create snowpack config snowpack.config.json

{ "scripts": { "run:js,html": "node --experimental-modules buildEntry.mjs", "run:js,html::watch": "nodemon -x $1", "mount:src": "mount app --to /", "mount:public": "mount app/public --to /", "mount:webmodules": "mount web_modules" }, "install": [ "@webformula/pax-core" ] }

Package.json scrips

The only scripts you'll need

Create package.json package.json

{ "scripts": { "start": "snowpack dev", "release": "snowpack build" } }
